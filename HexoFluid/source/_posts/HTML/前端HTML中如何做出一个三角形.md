```html
<!DOCTYPE html>
<html>
	<head>
		<meta charset="UTF-8">
		<title></title>
		<style type="text/css">
			div{
				width: 0;
				height: 0;
				border: 200px solid transparent;
				/*border-left-color: gold;*/
				/*border-right-color: pink;*/
				border-bottom-color: skyblue;
				
			}
		</style>
	</head>
	<body>
		<div></div>
	</body>
</html>

```
