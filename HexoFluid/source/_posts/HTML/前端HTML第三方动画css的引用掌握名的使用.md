```html
<!DOCTYPE html>
<html>
	<head>
		<meta charset="UTF-8">
		<title></title>
		<style type="text/css">
			div{
				width: 300px;
				height: 300px;
				background-color: pink;
			}
		</style>
		<link rel="stylesheet" type="text/css" href="css/animate.css"/>
	</head>
	<body>
		<div class="bounceInUp animated faster">
			
		</div>
	</body>
</html>

```
